===E4X Disable HTS===
This is a minimod for E4X. It may be used with Interregnum.

This minimod disables The HTS Faction. It may be used in combination with any other "Disable <SpecificFactionName>" minimods, as long as at least one faction is still available.

DO NOT use with "INT Disable Vanilla Factions".

This minimod does not change how the map generates, so random encounters or Minor Factions featuring Trader units may still appear.

---Compatible With---
Enhanced 4X Mod
Star Wars: Interregnum
Sins: Remastered
Most E4X/INT Minimods

---Not Compatible With---
INT Disable Vanilla Factions

---Installation---
To enable this mod, place this folder into your mod folder. Then, in the mods tab of the main menu, enable this mod
ABOVE the main Enhanced 4X Mod and Interregnum folders, if in use. 

For assistance with load order, see https://goafan77.github.io/E4XInterregnumModManager.html
