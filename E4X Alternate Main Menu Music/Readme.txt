===E4X Alternate Main Menu Music===
A minimod made for the Enhanced 4X Mod (http://forums.sinsofasolarempire.com/414639), but will work with any mod. It changes the main menu music to the Original Sins theme.

---Compatible With---
Enhanced 4X Mod
Star Wars: Interregnum
Sins: Remastered
All E4X Minimods
Most INT Minimods

---Not Compatible With---
INT Beskar

---Installation---
To Install, place this mod ABOVE the main Enhanced 4X Mod in your mod order. In the mods tab of the main menu, enable this mod ABOVE the main Enhanced 4X Mod, and ABOVE Interregnum if in use.

---Mod Load Order---
Other E4X/INT Minimods (optional)
E4X Alternate Main Menu Music
Other E4X/INT Minimods (optional)
Interregnum (optional)
Enhanced 4X Mod
Sins: Re (optional)

---Sample EnabledMods.txt---
TXT
Version 0
enabledModNameCount 3
enabledModName "E4X Alternate Main Menu Music"
