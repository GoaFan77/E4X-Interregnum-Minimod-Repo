---About---

This is a minimod for the Enhanced 4X Mod (http://forums.sinsofasolarempire.com/414639). It doubles the amount of fleet supply and capitalship slots available. This mod will work with other mods for the vanilla factions.

---Installation---

*Enable the minimod ABOVE the main Enhanced 4X Mod.

*Use this mod with the Deathmatch minimod to further increase fleet supply.

---Other---

For further information, see the readme in the main Enhanced 4X mod or visit the thread in the URL above.
