===INT Deathmatch Mode===
This is a minimod for Star Wars: Interregnum. If you are using Interregnum, use THIS minimod instead of the E4X version.

This minimod greatly increases income, culture, research speed, fleet supply etc. according the the speed settings.

In general, slow is the equivelent of unmodded normal, normal is equivelent of unmodded faster, and fast and faster are beyond normal Sins settings.

You may want to use the mod with the faster combat mod, for a truly fast paced game.

---Compatible With---
Enhanced 4X Mod
Star Wars: Interregnum
Sins: Remastered
Most E4X/INT Minimods

---Not Compatible With---
INT Beskar

---Installation---
To install the minimod, place this folder into your mod folder. In the mods tab of the main menu, enable this mod
ABOVE the main Enhanced 4X Mod and Interregnum. For best experience, this minimod should be loaded at the top of the minimod stack, below only INT SinsRe and INT All E4X Random Encounters if in use.

If you are using the resetting titan levels minimod, make sure this minimod is above it in the mod order.

For assistance with load order, see https://goafan77.github.io/E4XInterregnumModManager.html

---Mod Load Order---
INT SinsRe (optional)
INT All E4X Random Encounters (optional)
INT Deathmatch Mode
E4X Resetting Titan Levels (optional)
Other E4X/INT Minimods (optional)
Interregnum
Enhanced 4X Mod
Sins: Re (optional)
