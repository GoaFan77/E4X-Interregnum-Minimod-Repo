===INT Disable Vanilla Factions===
This is a minimod for Star Wars: Interregnum. DO NOT USE THIS MINIMOD with E4X.

This minimod disables all vanilla Sins/E4X factions, allowing you to similate battles from the Galactic Civil War.

This minimod does not change how the map generates, so random encounters featuring Vanilla units may still appear.

---Compatible With---
Enhanced 4X Mod
Star Wars: Interregnum
Sins: Remastered
All E4X/INT Minimods

---Installation---
To enable this mod, place this folder into your mod folder. Then, in the mods tab of the main menu, enable this mod
ABOVE the main Enhanced 4X Mod and Interregnum folders. 

---Mod Load Order---
Other E4X/INT Minimods (optional)
INT Disable Vanilla Factions
Other E4X/INT Minimods (optional)
Interregnum 
Enhanced 4X Mod
Sins: Re (optional)
